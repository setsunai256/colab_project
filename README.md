Генератор докладов и презентаций

Веб-приложение на Python (Flask) для автоматической генерации докладов и презентаций по заданной теме. Пользователь вводит тему, а приложение создаёт:

-Документ DOCX с текстом доклада
-Презентацию PPTX с тезисными слайдами
-Генеративные изображения для слайдов

Приложение использует модели генерации текста и изображений через Transformers и Diffusers, запускается в Colab и может работать с удалённым доступом через ngrok.

Технологии:

- Python 3.9+
- Flask
- Transformers
- Diffusers
- python-docx, python-pptx
- ngrok
- Google Colab

Быстрый запуск

1. Скопируйте весь код из full_project.py.
2. Вставьте в Colab или на локальный компьютер с Python 3.9+.
3. Запустите скрипт.
4. Откройте предоставленный URL ngrok в браузере и используйте интерфейс для генерации докладов и презентаций.

Использование

1. Введите тему в поле на главной странице.
2. Нажмите «Сгенерировать».
3. Скачайте DOCX и PPTX файлы с результатом.
